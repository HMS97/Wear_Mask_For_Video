# How to use


## 1. Clone this repo

```
git clone https://github.com/wuchangsheng951/Wear_Mask_For_Video
cd Wear_Mask_For_Video
```

## 2. Install requirements

```
pip install -r requirements.txt
```

## 3. Run the code

```
python app.py
```
